# Dice Roller App
This simple mobile application allows users to roll a random number on a cube. It provides various features such as selecting a cube, changing the result, adding a custom cube, and keeping a history of rolled values.
## Features
+	Random Number Generation: The app allows users to roll a random number on a cube by selecting one from the available options.
+	Cube Selection: Users can choose from a list of predefined cubes with different numbers of sides.
+	Result Modification: After rolling the cube, users have the option to modify the result manually if desired.
+	Custom Cube: The app provides the functionality to add a custom cube with a specific number of sides defined by the user.
+	History Tracking: The app keeps a record of all rolled values, allowing users to review their previous results.
## Technologies Used

+	Android SDK

+	Navigation Components
